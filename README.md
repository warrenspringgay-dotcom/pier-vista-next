# Pier Vista Fish & Chips website

Next.js App Router starter site for Pier Vista.

## Run locally

```bash
npm install
npm run dev
```

Open http://localhost:3000

## Deploy

Create a new Vercel project and import this folder/repo.

## Files to edit

- `src/app/page.tsx` — main website content/menu
- `src/app/layout.tsx` — SEO metadata
- `public/images/` — site images
